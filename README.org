* Description
A new version of a customised [[https://typeof.net/Iosevka/][Iosevka]] font, [[https://github.com/emacsomancer/iosevka-crown-of-stars][Crown of Stars]], a quasi-proportioned typeface Egyptian serifs and other aesthetic and visually functional specifications.

* Samples & Specimens
** Basic characters & symbols
Showing numerals, upper- and lower-case basic roman letters, a few accented/modified characters, and some symbols, in both upright and italics.
